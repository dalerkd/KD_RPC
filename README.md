# KD_RPC
## 用途
- 支持同步和回调

- 支持指针数据修改
将指针作为数据传输给远程，支持远程主机将数据修改结果回传在同步下。
支持形如下面的调用形式：

```
struct Argvs
{
  char* data;
  char data_len;
  float other;
}
int Function(Argvs* pArgvs,FARRPOC Callback);
```

## 优势
- 和本地调用没有差别的使用体验

流畅的接口，不需要额外显式初始化工作。

- 接口动态更新

可以对接口功能能按照需要进行动态更新。
